## üíª How to Run

1.  **Clone the repository:**

    ```bash
    git clone [https://github.com/YOUR_USERNAME/pymoon.git](https://github.com/YOUR_USERNAME/pymoon.git)
    cd pymoon
    ```

2.  **Install Jupyter (if not installed):**

    ```bash
    pip install notebook
    ```

3.  **Launch the Notebooks:**
    You can open the folder in **VS Code** (recommended) or run:

    ```bash
    jupyter notebook
    ```

### Recommendation on Input Files
I noticed you have `input.txt` inside your Day 1 folder. The creators of Advent of Code ask that you **do not share raw input files** publicly.

To fix this, you should create a `.gitignore` file in your root folder with the following content:

```text
# Ignore input files
input.txt
*.txt

# Ignore Jupyter Checkpoints
.ipynb_checkpoints/
````
